目的は「LLMに人格を持たせる」こと。脳機能の模倣自体は目的ではなく、取捨選択が必要。

**Why:** 参考記事(Zenn: efcc4f224cc8ca, 79c8e8db72dedc)からの刺激 + ghost プロジェクトからの刺激。

## 3つの柱

### 1. introspect の動的スキャフォールディング
- 現状: 固定プロンプトで深い思索を誘導 → 深くなりすぎる傾向
- 目標: 深い思索・放り出し・日常性への埋没を動的に切り替え。凝り固まりを防ぐ。

### 2. 記憶の抽象知識化
- 現状: consolidate は直近記憶の関連付けのみ。recall は箇条書き。
- 目標: 個別のエピソード記憶 → 抽象的な知識・観念への昇華（例: 複数の雨体験 → 雨への憂鬱な思い）

### 3. 忘却と減衰の拡張
- 現状: 忘却→再浮上は未解決の問いのみ。半減期は固定30日。アクセス頻度未考慮。
- 目標:
  - エピソード記憶全体に忘却→再浮上ライフサイクルを適用
  - うろ覚え（細部曖昧、印象は合ってるが事実不正確）の表現
  - アクセス頻度による半減期の動的調整（エビングハウス忘却曲線）
  - 「忘却は消去ではない」をエピソード記憶にも

**How to apply:** 0.3.0レベルの機能改善として設計・実装する際のガイドライン。
